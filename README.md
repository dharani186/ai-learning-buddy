 🤖 AI Buddy – Machine Learning Fundamentals Tutor

AI Buddy is an interactive AI-powered learning application that helps students understand **Machine Learning Fundamentals** through simple explanations, real-life examples, quizzes, and personalized feedback. The application is designed for beginners and intermediate learners, making Machine Learning easier to learn in an engaging and interactive way.


 📌 Project Overview

This project uses a Large Language Model (LLM) to create a virtual tutor named **Dharani**. The tutor adapts to the learner's knowledge level and learning goals, providing step-by-step guidance on Machine Learning concepts.

Whether you're preparing for exams, interviews, college coursework, or personal learning, AI Buddy offers an interactive learning experience with instant feedback and practice questions.



 ✨ Features

- 📚 Beginner-friendly explanations
- 🎯 Personalized learning based on user level
- 💡 Real-life examples and analogies
- 📝 Interactive quizzes with instant feedback
- 🤝 AI-powered conversational tutor
- 📖 Topic summaries
- 🚀 Beginner project suggestions

⚙️ Installation

 1. Clone the repository
    `bash
git clone https://github.com/your-username/AI-Buddy.git


### 2. Navigate to the project folder

```bash
cd AI-Buddy
```

### 3. Create a virtual environment (Optional)

Windows

```bash
python -m venv venv
venv\Scripts\activate
```

Linux / macOS

```bash
python3 -m venv venv
source venv/bin/activate
```

### 4. Install dependencies

```bash
pip install -r requirements.txt
```

### 5. Configure your API key

Create a file named **`.streamlit/secrets.toml`**

```toml
OPENAI_API_KEY="your_api_key_here"
```

Alternatively, use an environment variable.

## ▶️ Running the Application

```bash
streamlit run app.py
```

## 🎓 Learning Flow

1. Student enters their knowledge level.
2. AI explains Machine Learning concepts.
3. Real-life examples are provided.
4. Student answers practice questions.
5. AI evaluates responses and provides feedback.
6. Lesson ends with a quiz and summary.
